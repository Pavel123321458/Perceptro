Выводы
Однослойный перцептрон успешно классифицирует линейно разделимые данные (accuracy ~95%)

На чистых данных гиперпараметры (lr, batch_size) практически не влияют на итоговую точность

Большая инициализация весов (N(0, 10)) ухудшает сходимость из-за насыщения сигмоиды

Hinge loss и Binary Cross-Entropy дают сопоставимые результаты на линейно разделимых данных

L2-регуляризация не улучшает качество при отсутствии переобучения, но уменьшает норму весов

Momentum с β=0.9 ускоряет сходимость в начале обучения

Кросс-валидация подтверждает стабильность модели: стандартное отклонение точности не превышает 0.03

Основные ошибки — точки вблизи разделяющей границы, что ожидаемо для линейного классификатора
